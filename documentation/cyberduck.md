# How to access using Cyberduck desktop client

Info on how to access RENCI NWM hurricane data via Cyberduck.

# Steps

1. Obtain and install appropriate version of Cyberduck
* See https://cyberduck.io/ 
2. Use the add resource option, selecting AWS
3. Configure as follows:
* renc.osn.exsede.org
* private key
* secret key
